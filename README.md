# Verdantide

A turn-based open-world RPG built with Python, Pygame, and Tkinter.

## Features

- **Open World Exploration** - Explore 11 unique biomes using Pygame
- **Turn-Based Combat** - Strategic battles with elemental strengths/weaknesses
- **Biomes** - Forest, Prism Reef, Emberglow Caves, Drizzlewood, Mirrorsteppe, Nimbus Tundra, Obsidian Bloom Badlands, Riftglass Savannah, Sighing Mangrove, Aurora Drift, Hollowspire Jungle

## Installation

```bash
pip install pygame
```

## How to Play

```bash
python main.py
```

- **Movement**: WASD or Arrow Keys
- **Combat**: Select actions from the battle menu
- **Elements**: Fire > Ice > Wind > Fire

## Controls

- Navigate menus with mouse clicks
- WASD/Arrows to move in the open world
